WHAT IS KIRKI?

Kirki is a no-nonsence Toolkit allowing WordPress developers to use the Customizer and take advantage of its advanced features and flexibility.

Minimalistic and using the WordPress APIs to create beautiful and meaningful user experiences, it creates advanced controls for the customizer, takes care of the CSS output for you, creates postMessage scripts and saves you time while developing complex projects.

Don't worry about pricing, Kirki is 100% open-source and free. All development is done on github and has extensive documentation publicly available and editable!
